# LcL_Tools 工具集合

## 简介
LcL_Tools是一个基于3ds Max的工具集合插件，提供了便捷的建模和导入工具。

## 功能特性
- **FBX自动导入**: 一键导入指定路径的FBX模型文件

## 安装方法
1. 将`LcL_Tools.mzp`文件拖拽到3ds Max视窗中
2. 或者通过菜单: MAXScript > Run Script，选择`LcL_Tools.mzp`文件

## 使用方法
1. 安装后，图标会显示在工具栏中
2. 点击工具栏中的"LcL"按钮启动插件
3. 在"FBX导入工具"面板中，点击"导入FBX文件"按钮
4. 插件将自动导入`E:\Temp\output.fbx`文件

## 图标说明
- 插件使用自定义图标显示在工具栏中
- 图标文件位于`icon\small_icon\`目录下
